# Preguntados - Aplicación Web  
**Lenguajes de Programación - Tarea 1**

Este proyecto consiste en el desarrollo de una **aplicación web del juego Preguntados**, utilizando frameworks modernos para separar la arquitectura en **Frontend** y **Backend**.

Ambas partes trabajan de forma independiente pero se comunican entre sí para garantizar el funcionamiento completo de la aplicación.

---

## Descripción del Proyecto

La aplicación se desarrolla utilizando frameworks como técnica de desarrollo web.  
Esto permite dividir la aplicación en dos componentes principales:

### Frontend
Es la parte visual de la aplicación con la que interactúa el usuario.

### Backend
Es la parte encargada de la lógica del sistema, procesamiento de datos y manejo de la información.

Ambos componentes funcionan de forma independiente, pero interactúan entre sí para asegurar la funcionalidad de la aplicación.

---

## Tecnologías Utilizadas

### Backend
- Node.js  
- Express.js  
- JSON como almacenamiento de datos  
- CORS para la comunicación entre cliente y servidor  

### Frontend
- React  
- JavaScript  
- HTML5  
- CSS3  

---

## Instalación del Proyecto

Para ejecutar el proyecto en tu computadora sigue los siguientes pasos:

### 1. Clonar el repositorio

### 2. Ingresar a la carpeta del proyecto

cd Preguntados_AppWeb

### 3. Instalar dependencias

Abrir una terminal en la carpeta principal del proyecto y ejecutar:

npm run install-all
npm install concurrently

### 4. Ejecutar la aplicación
npm start
Ejecución

Al ejecutar el comando:

npm start

se iniciarán Frontend y Backend simultáneamente, permitiendo utilizar la aplicación web desde el navegador.
