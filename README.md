# Lenguajes_Tarea1_AplicacionWeb
Desarrllo de aplicacion web "Preguntados"

Descripción del problema

Se necesita crear una aplicación web utilizando los frameworks como técnica de desarrollo, donde nacen los términos de Frontend y Backend que tienen cualidades únicas y distintivas, actúan como unidades que, por separado, interactúan entre sí para asegurar la funcionalidad de una aplicación web. 

Para la interacción con el Frontend se va a utilizar la herramienta de react y Node js para el Backend. El desarrollo de la base de datos estará encargado por la herramienta de json. 

La aplicación web que desarrollada es una interacción de entretenimiento con el cliente donde se simula un juego de preguntados para determinar el conocimiento de un tema en específico con una serie de preguntas referentes y opciones de respuesta, al final se sabe el resultado de las preguntas, los aciertos y fallos.
