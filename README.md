# Lenguajes_Tarea1_AplicacionWeb
Desarrllo de aplicacion web "Preguntados"

#Desarrollo 

Se necesita crear una aplicación web utilizando los frameworks como técnica de desarrollo, donde nacen los términos de Frontend y Backend que tienen cualidades únicas y distintivas, actúan como unidades que, por separado, interactúan entre sí para asegurar la funcionalidad de una aplicación web. 


#Tecnologías utilizadas 

##Backend 

-Node.js 

-Express.js 

-JSON como almacenamiento de datos 

-CORS para comunicación entre cliente y servidor 

##Frontend 

-React 

-JavaScript 

-HTML5 

-CSS3 


#Instalación del programa

-En la carpeta principal del repositorio del proyecto Preguntados_AppWeb abrir consola e instalar
  *npm run install-all
  *npm start
